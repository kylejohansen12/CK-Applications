# C&K Applications Website

Professional pressure washing and roof coating services for North Carolina.

## Quick Start

1. Clone this repository
2. Open `index.html` in your browser
3. Deploy to GitHub Pages

## Features

- Responsive design
- Portfolio gallery
- Contact information
- Service listings

## Customization

Edit `index.html` to:
- Change phone number: `(919) 621-5375`
- Change email: `info@ckapplications.com`
- Update colors: `#003D7A` (blue), `#D4AF37` (gold)
- Add/remove portfolio items

## Deploy to GitHub Pages

1. Push to GitHub
2. Go to Settings → Pages
3. Select "main" branch
4. Your site is live!
